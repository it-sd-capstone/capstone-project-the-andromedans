##############
--Andromedan--
##############

Andromedan is a 2D shmup inspired by arcade classics like Galaxian. Gameplay involves surviving waves of basic enemies, after which
a boss fight occurs. The pattern of three basic enemy waves followed by a boss wave repeats indefinitely. Players are rewarded
for the amount of time they can survive and their total score earned during a run.


################
--Installation--
################

This project requires Unity. To install:
1. Navigate to https://unity.com/download

2. Download Unity Hub.

3. Double click on Unity Hub installer.

4. Follow installation steps on recommended settings.

5. Click Install.

6. Open Unity Hub.

7. On the left vertical toolbar, click on "Installs"

8. Click "Install Editor"

9. Select Unity 6.4 build 6000.4.3f1

10. Click "Continue"

To open Andromedan:
1. Open Unity Hub

2. On the left vertical toolbar, click on "Projects"

3. From the "Add" dropdown, select "Add project from disk"

4. Navigate to the directory containing Andromedan

5. Click "Open"

6. Select Andromedan from Unity's list of projects

--Usage--
#########

To deploy as executable:
1. Navigate to File > Build Settings.

2. Add MainMenu and Scene1 to the "Scenes in Build" list. 

3. Select "PC, Mac, and Linux Standalone" from the "Platform" list.

4. Select "Windows" from the "Target Platform" dropdown list.

5. Select "x86" from the "Architecture" dropdown list.

6. Press the "Build" button.

7. Select a directory and name the output file "Andromedan.exe"

8. Navigate to the directory created in step 7 and double click Andromedan.exe
